This website was developed using standard web technologies including:

  • HTML for semantic page structure
  • CSS for styling and responsive layout
  • JavaScript for interactivity and dynamic content
  • Chart.js for data visualization

The site includes pages for product display, inventory management (CRUD functionality), analytics dashboards, and more. I used modular code organization, folder structures (`css/`, `js/`, `img/`), and event-driven JavaScript to manage user interaction and product data.
